Here are all the assignments, {{currentUser.firstName}}. These are generated automatically, but changing them here will let you override that. Note that if a user has already been assigned to someone for a particular group, you will be unable to assign that user to someone else until you delete the first assignment. You'll also need to know the ID's of the groups and users if you want to create new ones, because an assignment is just a simple group ID, giver ID, and receiver ID (giver and receiver are both users). The names are just there for readability. If you just want to see the assignments for a particular group or user, you may prefer to navigate to that group page or user profile, respectively.
