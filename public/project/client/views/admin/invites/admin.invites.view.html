Invites

Here you can see all the invites users have yet to respond to. When a user accepts an invite, the user is added to the group they were invited to and the invite is deleted. Only group leaders and admins can create invites. Only one invite per user per group can exist at any given time (so you can't just send someone a bunch of invites to the same group). However, from this page you are able to create invitations to a group for users already in the group. This is impossible for group leaders to do because of the way they invite new users, but that restriction is not applied here. Try to avoid this, but know that a duplicate invite for a user won't cause them any real problems. If they're already in a group they'll stay in the group regardless of how they respond to the extra invitation.
